1. Introduction

"My Life in 2034" is a personal portfolio website focused on presenting my future self. The site covers various topics such as my daily life, hobbies, career, and sideline, with each topic represented by individual pages. It was created using HTML, CSS, and JavaScript, with interactive media elements such as images and videos.

2. Structure and Inspiration

The website consists of 5 main pages:

  • Homepage: A summary of my life and future vision.
  • Daily Life: Describes my routine, including cooking and exercise.
  • Career: A section about my work as a software developer.
  • Sideline: Highlights my activities as a YouTuber.
  • Hobbies: Covers my interests like reading, driving, and investing.

I was inspired by clean and intuitive portfolio websites, aiming for easy navigation and a quick way to explore each topic.

3. Accessibility

To make the website accessible for a variety of users, I incorporated several accessibility features:

  • All images include descriptive alt text for screen readers.
  • Clear navigation bars and menus help users easily find information.
  • The site uses a responsive layout, ensuring it works well across different devices.

4. Usability

The website was designed with user experience in mind:

  • The content is clearly divided by topic, making it easy for users to navigate.
  • Images and video sliders enhance the visual experience and user engagement.
  • A simple and intuitive UI improves usability and reduces confusion.

5. Lessons Learned

Through the development of this website, I gained valuable experience in:

  • Integrating various media elements like video and image sliders effectively.
  • Implementing responsive design across multiple devices.
  • Designing with accessibility in mind to create a more inclusive user experience.

6. Evaluation I: Successful Implementation

The most successful aspects of the project include:

  • Effective integration of image sliders and video elements.
  • A well-structured navigation system using reusable header and footer components.

7. Evaluation II: Areas for Improvement

Some areas that could be improved:

  • Consistency in design across all pages could be enhanced.
  • The slider animations could be made smoother and more interactive.

8. Resources

The project utilized several external resources:

  • Fonts from Google Fonts (Lilita One, Montserrat).
  • JavaScript for image sliders and video elements.
